Description
------------

fb_remote est un outil en ligne de commande pour simuler la télécommande de la Freebox.

Usage
------

::

  fb_remote touche[+] [nombre_de_repetition]


Noms des touches
-----------------

- red
- green
- yellow
- blue
- power
- list
- tv
- 0 à 9
- back
- swap
- info, mail, help, pip
- epg, media, options
- vol_inc, vol_dec
- prgm_inc, prgm_dec
- ok
- up, right, down, left
- mute
- home
- rec
- bwd
- prev
- play
- fwd
- next

Un nom de touche suivi de "+" indique un appui long.
